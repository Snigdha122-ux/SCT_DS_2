# Titanic Dataset Exploratory Data Analysis (EDA)

## Objective

Perform data cleaning and exploratory data analysis on the Titanic dataset to identify patterns and relationships among variables.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

Titanic Dataset from Kaggle

## Data Cleaning

* Filled missing Age values using median
* Filled missing Embarked values using mode
* Removed Cabin column due to excessive missing values

## Exploratory Data Analysis

* Survival Count Analysis
* Survival by Gender
* Age Distribution
* Passenger Class Distribution
* Correlation Heatmap

## Key Findings

* Females had higher survival rates than males.
* First-class passengers had better chances of survival.
* Most passengers were young adults.
* Passenger class influenced survival probability.

## Conclusion

EDA helped identify important patterns and relationships in the Titanic dataset using statistical summaries and visualizations.
